Conectar Host - Vultr

Configurações de Conexão

Instruções:
  1. Acesse Vultr API Settings.
  2. Verifique a lista Access Control. Ela DEVE conter a regra 0.0.0.0/0 (Todas as redes IPv4).
  3. Se não tiver o botão "Allow All IPv4", adicione manualmente a rede 0.0.0.0 com máscara 0.
  4. Recomendado: Clique também em "Allow All IPv6" (::/0) para evitar bloqueios se a Vercel usar IPv6.
  5. Copie a API Key e cole abaixo.
